安装express

npm install express

创建server.js. 引入express,并实例化

    const express = require('express');
    const app = express();
    const port = process.env.PORT || 5000;

app.listen(port,() => {
    console.log(`Server running on port ${port}`);
})

运行 node server.js可以看到打印

添加路由使其可以在浏览器访问

app.get("/", (req, res)=>{
    res.send("Hello Word")
})

重启服务

为了避免重启，引入nodemon

studo npm install nodemon -g
执行 nodemon server.js

在package.json中添加启动命令

"scripts": {
    "test": "echo \"Error: no test specified\" && exit 1",
    "start": "node server.js",
    "server": "nodemon server.js"
  },
