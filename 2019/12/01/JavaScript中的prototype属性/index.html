定义

prototype 属性允许您向对象添加属性和方法，Prototype 是全局属性，并适用于所有的 Javascript 对象。

用法

object.prototype.name=value

注意

prototype属性是函数所独有的，由于在js中一切皆对象，函数也不例外，故这并非和其定义相矛盾。
