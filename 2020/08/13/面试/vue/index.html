Vue

  • 在vue中watch和created哪个先执行？为什么？
    • computed-create-mounted 如果watch 加了 immediate: true, 就是watch先执行，否则就是created 先执行
  • vue中mixins和extends, extend有什么区别？
    • extend用于创建vue实例; mixins可以混入多个mixin，extends只能继承一个；
    • mixins类似于面向切面编程(AOP), extends 类似于面向对象编程
    • 优先级：vue.extend > extends > mixons
  • vue中mixins有什么使用场景？
    • 如果只是提取公共的数据或者通用方法，并且他们不需要组件之间进行维护
  • 在vue中created与activated有什么区别？
    • activated是组件被激活后的钩子函数，每次回到页面都会执行
    • created在创建vue对象时，当html渲染之前就触发；但是注意，只会触发一次
    • 执行顺序：created => mounted =>activated
  • 在vue项目如何引入异步组件？https://www.jianshu.com/p/7c848df61801
    • ① 主要是应用了component的 is 属性
    • ② 借助vue的动态组件和webpack打包单文件来实现动态加载异步组件，通过vue的事件总线挂载在this.$root上来实现平级组件之间的通信。
    • ③ 在路由层面懒加载
  • 在vue项目中scss scoped穿透符>>>无效的解决方案有哪些？
    • 可以使用 /deep/ 操作符( >>> 的别名)
    • 官方介绍可以使用 ::v-deep https://vue-loader.vuejs.org/guide/scoped-css.html#deep-selectors
  • 为什么在v-for中的key不推荐使用随机数或者index呢？那要怎么使用才比较好呢？
    • 会影响diff判断 用id唯一值比较好
  • vue-loader在webpack编译流程中的哪个阶段？ https://blog.csdn.net/DiDi_Tech/article/details/100913050
    • vue-loader允许我们在 SFC(single file component) 中去写我们的 template/script/style
