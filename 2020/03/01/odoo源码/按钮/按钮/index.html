    属性

    * icon  可用的icon在 addons/web/static/src/img/icons
* string 按钮的显示文字
* type  值可以是 workflow, object action，   默认是 workflow
* name  就是要触发的方法标识
* args  传递方法的参数
* content 上下文
* confirm  针对对话框的确认
* special="cancel" 用于向导
* states 可见的状态
* classname 加载的类名(常用 oe_highlight)
